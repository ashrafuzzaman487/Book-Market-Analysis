# 📊 Book Market Analysis & Inventory Automation

This project focuses on automated data extraction and professional reporting. By scraping book market data, the system processes raw information into structured datasets and generates comprehensive analytical reports.

## 🚀 Project Workflow
1. **Scraping**: Automated data collection from online book marketplaces.
2. **Data Cleaning**: Processing and transforming raw data into an inventory-ready format.
3. **Insight Generation**: Analyzing the market trends to produce professional reports.

## 📂 Project Assets
- `filter_books.py`: The core script for data collection and cleaning.
- `Cleaned_Book_Inventory.pdf`: A detailed document showing the cleaned and structured book inventory.
- `Book_Market_Insights_Report.pdf`: A comprehensive report providing analytical insights and market trends.

## 🛠️ Key Technical Skills
- Python (Data Processing)
- Web Scraping (BeautifulSoup/Requests)
- Data Cleaning & Logic implementation
- Automated Reporting (PDF Generation)

## 💡 How to Use
1. Download the files from this repository.
2. Run the `filter_books.py` script to update the data.
3. Review the `/PDF_Reports` for detailed insights and market trends.
